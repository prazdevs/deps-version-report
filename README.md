## Usage

### By installing the package

1. Install the package with `yarn add -D deps-version-report` or `npm i -D deps-version-report`.
2. Add script to `package.json` : `deps-version-report [--path <package.json>] [--dest <report.csv>]`.

### Through npx

1. Simply run `npx deps-version-report [--path <package.json>] [--dest <report.csv>]`
